# CodeAlpha - E-Commerce Store

## Project Overview

This is a full-stack E-Commerce Store developed as part of the CodeAlpha Internship. The project allows users to browse products, register, log in, add products, and place orders using a MongoDB database.

## Features

- User Registration
- User Login
- Product Listing
- Product Details
- Shopping Cart
- Order Processing
- MongoDB Database Integration
- Responsive Frontend
- Express.js Backend

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/codealpha_Ecommercestore.git
```

2. Install dependencies

```bash
npm install
```

3. Start MongoDB

Make sure MongoDB is running on your system.

4. Run the project

```bash
node server.js
```

5. Open in your browser

```
http://localhost:5000
```

## Database

MongoDB is used to store:
